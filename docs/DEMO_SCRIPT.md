## Common Questions

**Q: Does it work offline?**
> Basic operations continue. Full sync when internet returns.

> Every action is logged with timestamp. You have full audit trail.

**Q: What about OTA bookings (Booking.com, MakeMyTrip)?**
> Mark booking source when creating. OTA sync coming in future update.

**Q: How is data secured?**
> Industry-standard encryption. Each hotel's data is isolated.
